FRAGFARM Mobile Web

Mobile-First E-commerce Website

브랜드 정체성과 사용자 경험이 자연스럽게 연결되는 모바일 퍼스트 기반의 이커머스 홈페이지 재설계 프로젝트

01 Project Overview

  • Role

    UI Design · Front-End Development

  • Tools

    Figma · HTML5 · CSS3 · JavaScript

  • Duration

    2026.01~2026.06

02 Project Goal

  • Brand Identity

    브랜드 정체성을 반영한 모바일 UI 구조 설계

  • User Flow

    사용자 흐름 중심 정보 구조 개선

  • Brand Experience

    브랜드 감성과 사용성의 자연스러운 연결

03 Design Concept

  • Unified
  • Effortless
  • Flow

자연의 흐름에서 출발한 브랜드 무드를 기반으로 직관적인 인터페이스와 이미지 중심 레이아웃을 통한 브랜드 경험과 사용성의 자연스러운 통합

04 Key Features

  • Brand-Driven Icon Navigation

    상태 기반 아이콘 전환을 통한
    브랜드 경험 강화 네비게이션

  • Section-Based Main Layout

    주요 카테고리를 빠르게 탐색하고 확장 탐색을 유도하는
    섹션 기반 메인 레이아웃

  • Product Browsing Control

    1열/2열 보기 전환과 페이지 분리를 통한
    탐색 효율 향상

  • Quick Shopping Interaction

    상세 페이지 이동 없이 위시리스트·장바구니로
    빠른 쇼핑 지원
